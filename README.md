# Search Files Project v1.0

  ## Descripcion(español)
  El programa *Search Files* es un programa de busqueda de archivos y documentos en todo un equipo.
  Utiliza una interfaz grafica agradable y tanto moderna como facil de usar.
  Esta hecho para que lo utilize toda clase de publico.
  
  ## v1.0
  Es la primera y probablemente ultima version de este programa la cual esta equipada con todo lo necesario y con un procesamiento rapido.

  - Git y Github(Control de versiones)

  ## Description (English)
  The *Search Files* program is a file and document search tool that scans the entire system.
  
  It features a pleasant graphical interface that is both modern and easy to use.
  
  It is designed to be accessible for all types of users.
  
  ## v1.0
  This is the first—and likely final—version of the program, equipped with all necessary features and fast processing.
